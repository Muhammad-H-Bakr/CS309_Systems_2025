What is JavaScript?

JavaScript is the programming language of the web. It makes websites interactive and dynamic, allowing you to:

  • Change HTML content and attributes
  • Modify CSS styles dynamically
  • Respond to user interactions (clicks, inputs, etc.)
  • Validate forms and process data
JavaScript is not the same as Java. Despite the similar name, they are completely different programming languages!

1 Variables

Variables are containers for storing data values. JavaScript uses three keywords to declare variables:

  • var - older way to declare variables (avoid in modern JavaScript)
  • let - declares a block-scoped variable that can be changed
  • const - declares a block-scoped constant that cannot be reassigned
let name = "Alice";
const age = 25;
var city = "New York"; // Older method
// Using variables
console.log("Hello, " + name); // Output: Hello, Alice
Always use let or const instead of var in modern JavaScript!

3 Operators

Operators are used to perform operations on variables and values:

Arithmetic Operators

  • + Addition
  • - Subtraction
  • * Multiplication
  • / Division
  • % Modulus (remainder)
  • ++ Increment
  • -- Decrement

Comparison Operators

  • == Equal to
  • === Equal value and type
  • != Not equal
  • !== Not equal value/type
  • > Greater than
  • < Less than
// Arithmetic operations
let x = 10;
let y = 5;
console.log(x + y);  // 15
console.log(x - y);  // 5
console.log(x * y);  // 50
console.log(x / y);  // 2
console.log(x % y);  // 0
// Comparison operations
console.log(x > y);   // true
console.log(x === y); // false

4 Conditionals

Conditionals execute different code based on different conditions:

// if statement
if (condition) {
  // code to execute if condition is true
}
// if...else statement
if (condition) {
  // code if true
} else {
  // code if false
}
// Example
let age = 18;
if (age >= 18) {
  console.log("You are an adult");
} else {
  console.log("You are a minor");
}

6 Functions

1 Arrays and Array Methods

Arrays are used to store multiple values in a single variable. JavaScript provides many built-in methods to manipulate arrays:

Creating and Accessing Arrays

let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // "apple"
console.log(fruits.length); // 3

Common Array Methods

  • push() - Add element to end
  • pop() - Remove last element
  • shift() - Remove first element
  • unshift() - Add element to beginning
let colors = ["red", "green"];
colors.push("blue");     // ["red", "green", "blue"]
colors.pop();           // removes "blue"
colors.unshift("yellow"); // ["yellow", "red", "green"]
colors.shift();         // removes "yellow"

2 Array Iteration Methods

JavaScript provides powerful methods to iterate through arrays and transform data:

Method Purpose Returns
map() Creates new array with results of calling a function for every element New array
filter() Creates new array with elements that pass a test New array
reduce() Reduces array to a single value Single value
find() Returns first element that passes a test Element or undefined
forEach() Executes a function for each array element undefined 
let numbers = [1, 2, 3, 4, 5];
// map - create new array with doubled values
let doubled = numbers.map(x => x * 2);
console.log(doubled); // [2, 4, 6, 8, 10]
// filter - create new array with even numbers
let evens = numbers.filter(x => x % 2 === 0);
console.log(evens); // [2, 4]
// reduce - calculate sum of all numbers
let sum = numbers.reduce((acc, curr) => acc + curr, 0);
console.log(sum); // 15

3 Objects and Object Methods

Objects are collections of key-value pairs. They are fundamental to JavaScript and used to store structured data:

Creating and Accessing Objects

// Object literal
let person = {
  name: "Alice",
  age: 30,
  city: "New York"
};
// Accessing properties
console.log(person.name);     // "Alice"
console.log(person["age"]);   // 30

Modifying Objects

// Adding properties
person.job = "Developer";
// Modifying properties
person.age = 31;
// Deleting properties
delete person.city;

4 Object Methods

JavaScript provides several built-in methods to work with objects:

Method Description Example
Object.keys() Returns array of object's own enumerable property names Object.keys(obj)
Object.values() Returns array of object's own enumerable property values Object.values(obj)
Object.entries() Returns array of object's own enumerable property [key, value] pairs Object.entries(obj) 
let user = {
  name: "Alice",
  age: 30,
  city: "New York"
};
// Get keys
console.log(Object.keys(user));
// Output: ["name", "age", "city"]
// Get values
console.log(Object.values(user));
// Output: ["Alice", 30, "New York"]
// Get entries
console.log(Object.entries(user));
// Output: [["name", "Alice"], ["age", 30], ["city", "New York"]]

5 Destructuring

Destructuring allows you to unpack values from arrays or properties from objects into distinct variables:

Array Destructuring

let colors = ["red", "green", "blue"];
// Destructuring assignment
let [first, second, third] = colors;
console.log(first);  // "red"
console.log(second); // "green"
// Skip elements
let [primary, , tertiary] = colors;
console.log(tertiary); // "blue"

Object Destructuring

let person = { name: "Alice", age: 30, city: "NYC" };
// Destructuring assignment
let { name, age } = person;
console.log(name); // "Alice"
console.log(age);  // 30
// Assign to new variable names
let { name: fullName, age: years } = person;

6 Spread and Rest Operators

The spread (...) and rest (...) operators provide powerful ways to work with arrays and objects:

Spread Operator

Expands an iterable (like an array) into individual elements

// Array spreading
let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let combined = [...arr1, ...arr2];
console.log(combined); // [1, 2, 3, 4, 5, 6]
// Object spreading
let obj1 = { a: 1, b: 2 };
let obj2 = { c: 3, d: 4 };
let merged = { ...obj1, ...obj2 };
console.log(merged); // { a: 1, b: 2, c: 3, d: 4 }

Rest Operator

Collects multiple elements into a single array

// Function parameters
function sum(...numbers) {
  return numbers.reduce((a, b) => a + b, 0);
}
console.log(sum(1, 2, 3, 4)); // 10
// Array destructuring
let [first, ...rest] = [1, 2, 3, 4];
console.log(first); // 1
console.log(rest);  // [2, 3, 4]
